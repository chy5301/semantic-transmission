## 评估重点

1. **语义通信框架**：是否有端到端可运行的开源方案？
2. **ComfyUI 生态**：
   - API 模式的调用方式和限制
   - 文生视频/图生图的成熟工作流
   - 自定义节点的扩展机制
3. **论文配套代码**：核心论文是否有开源实现？代码质量和可运行性如何？
