## 对比维度

论文间对比时关注：

1. **传输内容**：传输什么（纯文本 / 文本+条件图 / 特征向量）？各自的码率是多少？
2. **还原质量**：客观指标（PSNR/SSIM/LPIPS）和主观质量如何？
3. **系统架构**：端到端训练 vs 模块化拼接？是否需要专用编解码器？
4. **实时性**：能否满足视频传输的实时性要求？
5. **泛化能力**：对不同场景、分辨率的适应性如何？
